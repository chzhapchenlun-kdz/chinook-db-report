项目名称：Chinook SQLite 数据库实践
	1.	数据集说明
本项目使用公开示例数据库 Chinook（SQLite 版本），模拟数字音乐商店的业务数据，包括 Artist、Album、Track、Customer、Invoice、InvoiceLine 等表。
	2.	实验环境

	•	数据库：SQLite（Chinook_Sqlite.sqlite）
	•	工具：Ridill SQLite（macOS）

	3.	主要查询内容

	•	艺术家专辑数量 Top 10（Q4）
	•	客户消费 Top 10（Q9）
	•	曲目收入 Top 10（Q10）
	•	艺术家收入 Top 10（Q11）
	•	按月份统计销售额（Q12）
	•	从未被购买的曲目数量（Q15）

	4.	文件说明

	•	sql/queries.sql：SQL 查询脚本
